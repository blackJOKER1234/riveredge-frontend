# 快财务 (kuaicaiwu)

## 模块说明
财务管理模块，处理应收应付、凭证账务等财务核心业务。

## 路由前缀
`/apps/kuaicaiwu`

## 功能模块
- 应收管理
- 应付管理
- 凭证管理
- 账务处理
- 财务报表

## 目录结构
- `components/`: 子应用专用组件
- `pages/`: 页面组件
- `services/`: API 服务
- `utils/`: 工具函数
- `types/`: 类型定义
- `constants/`: 常量配置

## 开发要点
1. 金额计算需注意精度处理（BigDecimal / 分转元等）
2. 财务日期、会计期间等概念需与后端保持一致
3. 凭证和账务操作涉及数据一致性，注意事务边界
4. 报表导出功能注重格式和性能
