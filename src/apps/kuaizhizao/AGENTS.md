# 快制造 (kuaizhizao)

## 模块说明
车间执行与协同核心——MES 制造执行系统。
工厂运营全流程管理，覆盖销售、采购、生产、仓储、质量、设备全链条。

## 路由前缀
`/apps/kuaizhizao`

## 功能模块
- **销售管理**: 客户池、报价、客户跟进、销售合同、销售预测、销售订单、发货通知、销售退货
- **采购管理**: 采购订单、采购入库、采购退货
- **仓库管理**: 生产领料、生产退料、成品入库、其他出入库
- **生产管理**: 生产计划、工单、工序报工、委外加工
- **质量管理**: 来料检验、过程检验、成品检验
- **设备管理**: 设备台账、模具台账、设备故障、维修保养计划
- **工程管理**: BOM、工艺路线、图纸文档
- **报表中心**: 多维度统计分析报表

## 目录结构
- `components/`: 子应用专用组件（如物料选择、BOM 展示、工艺路线配置等）
- `pages/`: 各功能模块页面
- `services/`: API 服务（manufacturing-specific）
- `hooks/`: 业务 Hooks
- `utils/`: 工具函数
- `types/`: 类型定义
- `constants/`: 模块常量
- `schemas/`: 数据校验 schema
- `routePrefetch.ts`: 路由预取

## 开发要点
1. 物料、BOM、工艺路线等核心领域概念需保持类型安全
2. 单据流转（报价→订单→工单→入库）链路复杂，注意数据一致性
3. 权限控制细致（按操作+单据类型），使用 manifest 中的 permission 字段
4. 与 master-data（物料、客户等主数据）解耦，通过 API 调用获取主数据
5. 报表查询注意大数据量场景下的性能优化
