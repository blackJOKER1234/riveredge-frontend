# 快AI (kuaiai)

## 模块说明
AI 相关功能模块，集成智能助手、自动化能力。

## 路由前缀
`/apps/kuaiai`

## 功能模块
- AI 助手
- 智能分析
- 自动化流程

## 目录结构
- `components/`: AI 专用 UI 组件（对话窗口、结果展示等）
- `pages/`: 页面
- `services/`: AI 服务 API
- `hooks/`: AI 相关 Hooks

## 开发要点
1. AI 对话类 UI 注意流式渲染和状态管理
2. 与后端 AI 服务的通信需处理超时和重试
3. AI 生成内容需有合适加载态和错误处理
4. 注意用户隐私和数据安全
