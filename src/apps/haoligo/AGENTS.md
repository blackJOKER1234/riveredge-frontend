# 好立购 (haoligo)

## 模块说明
采购商城模块，提供供应商管理和采购协同功能。

## 路由前缀
`/apps/haoligo`

## 功能模块
- 供应商管理
- 采购寻源
- 采购协同
- 商城管理

## 目录结构
- `components/`: 子应用专用组件
- `pages/`: 页面
- `services/`: API
- `hooks/`: Hooks
- `utils/`: 工具函数
- `constants/`: 常量
- `layouts/`: 子应用布局
- `styles/`: 子应用样式

## 开发要点
1. 商城类 UI 注重用户体验和移动端适配
2. 供应商准入和认证流程涉及多步骤表单
3. 采购协同涉及多方交互，注意通知和消息机制
