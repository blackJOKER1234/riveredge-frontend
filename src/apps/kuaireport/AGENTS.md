# 快报表 (kuaireport)

## 模块说明
报表系统，提供自定义报表、数据分析和可视化功能。

## 路由前缀
`/apps/kuaireport`

## 功能模块
- 报表设计器
- 报表查看
- 数据分析
- 可视化图表

## 目录结构
- `components/`: 报表组件（设计器、渲染器、图表组件等）
- `pages/`: 页面
- `services/`: API
- `utils/`: 工具函数
- `types/`: 类型定义

## 开发要点
1. 报表设计器为拖拽式交互，注意 DnD 性能和状态管理
2. 大数据量报表注意虚拟滚动和分页
3. 图表组件使用合适的可视化库，注意按需加载
4. 导出（PDF/Excel）功能保持格式兼容性
