# 主数据 (master-data)

## 模块说明
主数据管理模块，管理企业核心基础数据。

## 路由前缀
`/apps/master-data`

## 功能模块
- 物料主数据
- 客户主数据
- 供应商主数据
- 组织架构
- 编码规则

## 目录结构
- `components/`: 子应用专用组件
- `pages/`: 页面
- `services/`: API
- `hooks/`: Hooks
- `utils/`: 工具函数
- `types/`: 类型定义
- `constants/`: 常量
- `schemas/`: 数据校验 schema
- `routePrefetch.ts`: 路由预取

## 开发要点
1. 主数据是所有业务子系统的数据基础，修改需谨慎
2. 数据导入导出功能常见，注意大数据量和大文件处理
3. 编码规则模块需提供灵活配置界面
4. 物料等主数据维度多，注意高级搜索和筛选功能
